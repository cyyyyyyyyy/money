# A 股 ETF 多信号仓位轮动策略

这是一个周频 A 股 ETF 单仓回测项目。策略用规则化市场数据代理“大盘表现、市场噪声、风险偏好和主流情绪”，再决定每期只持有 1 只 ETF。

初版不直接持有国债 ETF，但会用国债 ETF 相对强弱作为风险偏好信号。
新闻和政策情绪可以通过 CSV 可选接入，和市场信号一起影响总分；没有输入时默认中性分 `50`。政策分权重高于普通新闻分，事件影响默认按 8 周逐步衰减回中性，避免一次政策新闻永久影响后续回测。

## 默认标的

| 角色 | 标的 |
| --- | --- |
| 大盘代理 | `000300` 沪深 300 |
| 行业轮动池 | `159819` 人工智能 ETF, `512480` 半导体 ETF, `159770` 机器人 ETF, `516160` 新能源 ETF, `561560` 电力 ETF |
| 观察池 | `563530` 卫星 ETF |
| 防守池 | `515100` 红利低波 100 ETF, `512800` 银行 ETF |
| 债券信号 | `511010` 国债 ETF |
| 成长扩散信号 | `588000` 科创 50 ETF, `159949` 创业板 50 ETF |
| 托底确认信号 | `510300` 沪深 300 ETF, `510050` 上证 50 ETF, `510500` 中证 500 ETF, `159915` 创业板 ETF |

## 当前因子

行业轮动池的 ETF 选择使用多因子打分：

- 4 周动量：短期趋势确认，避免过度依赖旧趋势。
- 8 周动量：中短期趋势延续。
- 12 周波动：波动越低越加分，用来过滤噪声。
- 12 周回撤：离阶段高点越远越扣分，避免弱势反弹误判。
- 成交热度：成交额放大时加分。
- 行业广度：行业轮动池里有多少 ETF 高于 20 周均线，用于判断进攻是否扩散。
- 成长扩散：科创 50 和创业板 50 相对沪深 300 的强弱，仅作为诊断输出，不直接改变持仓池。
- 托底确认：宽基 ETF 弱市放量且银行/红利相对强时，小幅提高市场状态分；该因子只做风险状态确认，不决定买入哪个行业。

## 运行

```bash
uv run money-strategy backtest --start 2018-01-01 --end 2026-05-12
```

默认参数偏进攻：

- 同类换仓门槛：`0.15`
- 进攻档阈值：`60`
- 防守档阈值：`35`
- 均衡档转防守阈值：`45`
- 档位确认：`1` 周
- 新闻权重：`0.03`
- 政策权重：`0.05`
- 主题权重：`0.06`

```bash
uv run money-strategy backtest
```

可选接入新闻/舆情分：

```bash
uv run money-strategy backtest --sentiment-file data/sentiment.csv
```

项目内置了一份基于官方来源整理的政策/新闻事件样例：

```bash
uv run money-strategy backtest --sentiment-file data/policy_events.real.csv
```

稳定执行最新信号：

```bash
scripts/run_stable_signal.sh
```

也可以指定结束日期：

```bash
scripts/run_stable_signal.sh 2026-05-12
```

该命令会输出并写入 `output/stable_signal/latest_signal.json`。

`data/sentiment.csv` 格式：

```csv
date,news_score,policy_score,theme_score_159819,theme_score_512480,theme_score_159770,theme_score_516160,theme_score_561560,note
2026-05-08,65,75,75,70,70,60,55,政策预期和主流报道偏积极
2026-05-12,45,50,55,55,50,50,50,外部扰动和风险偏好回落
```

主题分列是可选的，格式为 `theme_score_<ETF代码>`；提供后会小幅影响对应 ETF 的轮动因子分。
新闻热点不会在每次运行时无记录自动改仓；需要先写入 CSV，回测和实盘信号都会留下同一套事件输入。

输出会写入 `output/`：

- `weekly_signals.csv`：每周信号、分数、目标档位、目标权重
- `trades.csv`：每次单仓切换的买入日期、卖出日期、卖出原因、成交假设价格和持有收益
- `equity_curve.csv`：策略净值和基准净值
- `performance.csv`：年化收益、最大回撤、夏普、Calmar、换手率等
- `equity_curve.png`：净值曲线

```bash
uv run pytest
```

## 说明

本项目只用于策略研究和回测，不构成投资建议。实盘前需要继续验证数据质量、成交可行性、交易成本、滑点和不同市场阶段稳定性。
当前交易明细按周频收盘价回测口径记录；实盘执行建议固定为周信号确认后的下一交易日开盘或条件单执行，并单独记录真实成交价。
